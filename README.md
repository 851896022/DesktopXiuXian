## 食用方法

### 修炼

放置→增加修为

点击→增加修为

每日凌晨0点到6点修为翻倍！

其中凌晨2点到4点修为5倍！

### 操作


界面右键↓
结束修炼→退出程序

自废修为→删档

排行榜↓



###更换皮肤

右键→变身



都说变身没法改变大小，按住Ctrl+滚轮就可以啊！！！

### 华山论剑

右键->华山论剑




半透明＋置顶=摸鱼绝配
